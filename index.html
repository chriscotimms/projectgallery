Game Project

A simple Javascript memory-based game, with and additional effort to implement increased access, something missing in a lot of beginner javascript example games. Keyboard navigation using Tab and Arrow keys for focus, and Enter or mouseclick to activate choices. Aria-labels are assigned to all the images, as are large-font numbers. Aria-live politeness settings are high for the game area, and lower for the updating score.
